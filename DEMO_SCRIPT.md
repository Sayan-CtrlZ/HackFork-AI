## Fallback Protocol (If Live Demo Fails)

If gitclaw fails to connect or API is slow:

1. Say calmly: *"While that loads — let me show you what the output looks like."*
2. Open `PROJECT_BRIEF.md` — *"This is what ideate generated."*
3. Open `JUDGE_REPORT.md` — *"This is the adversarial self-evaluation."*
4. Open `PITCH.md` — *"And this is what the pitch skill wrote."*
5. Say: *"These five files are in the repo right now. The pipeline already ran. You can clone it and verify."*

The fallback is almost as strong as the live demo because all artifacts are pre-generated.
