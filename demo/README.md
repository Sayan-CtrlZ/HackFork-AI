# Demo: Sample Hackathon Run

This folder contains a **canned demonstration** of HackFork running on a sample project idea.
Use this to show judges the full pipeline output without needing a live API call.

## The Sample Idea
> "A CLI tool that watches your git commits and auto-generates a standup update for you."

## Output Files in This Folder
- `sample-PROJECT_BRIEF.md` — Output from ideate skill
- `sample-JUDGE_REPORT.md` — Output from evaluate skill (with Judge Mode scoring)
- `sample-PITCH.md` — Output from pitch skill

## How to Use in a Live Demo
1. Open `sample-PROJECT_BRIEF.md` — show what ideate generates from a vague idea
2. Open `sample-JUDGE_REPORT.md` — show the adversarial Judge Mode evaluation
3. Open `sample-PITCH.md` — show the auto-generated pitch with coached Q&A

These represent a complete pipeline run on a different project than HackFork itself,
proving the skills are general-purpose — not just self-referential.
