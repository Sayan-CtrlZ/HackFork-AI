## The Problem

Every developer on your team spends 5 minutes every morning trying to remember what they did yesterday — then writes a standup message that's either too vague to be useful or too technical for anyone to understand.

The data is sitting in git, timestamped, complete. It's just not in a format anyone reads.

---

## The Solution

**StandupBot**: Run one command in your git repo. In 8 seconds, you have a ready-to-paste Slack standup message summarizing the last 24 hours of commits in plain, conversational English — in your team's voice.

Zero forms. Zero memory. Just: `standupbot --repo .`

---

## The Demo Moment

1. Open a terminal in any git repo
2. Show the raw `git log --since=24h` — commits like "fix: race condition in auth middleware", "refactor: BaseServiceImpl cleanup", "chore: bump deps"
3. Run `standupbot --repo .`
4. 8 seconds later: *"Fixed a login bug causing intermittent failures. Cleaned up the core service layer. Updated dependencies to latest stable."*

The agent didn't summarize text. It translated your technical git history into human-readable English — automatically.

## Likely Judge Questions & Coached Answers

**Q: "This has been built before — what's different?"**

> Every standup tool starts by asking you to fill out a form. We start by reading your git history — the actual, timestamped record of what you built. That's not a UI improvement, it's a fundamentally different assumption about where the data lives.

**Q: "What if commits are vague or poorly written?"**

> That's a feature, not a bug. StandupBot will flag vague commits ("I noticed 3 commits just say 'fix'. Your team can't use this standup.") and suggest better messages. It makes bad commit hygiene visible — which is a conversation worth having.

**Q: "Does it integrate with Slack?"**

> Not yet — and deliberately not for this hackathon. The output goes to your clipboard and stdout. Paste it where you want. Adding Slack API integration is a 2-hour addition that removes zero-setup from our demo. We're protecting the demo moment.
